/*
Демонстрационная программа "Arinc429".
Программа конфигурирует тактовые сигналы, порты ввода/вывода,
передатчики первого и второго каналов интерфейса ARINC429,
приемники 1, 2, 3 и 4 каналов интерфейса ARINC429,
системный таймер, разрешает прерывания от системного таймера,
далее зацикливается.
По истечении времени TIMEOUT в секундах по первому каналу
в поле данных отправляется текущее значение переменной Sec,
в поле метки - всегда отправляется 1;
Принятые данные обрабатываются подпрограммой обработки прерываний,
в которой данные из FIFO приемника переносятся в FIFO второго
канала передатчика и отправляются обратно, а также проверяется
наличие ошибок при приеме.

Настройки:
тактовая частота контроллера HCLK 80МГц
частота прерываний 100мс
бит паритета дополняет до нечетного для 1 и 2 каналов
скорость обмена данными определяется определением LOWSPEED (см. файл config.c);
если LOWSPEED = 1, то устанавливается низкая скорость (12,5кГц), если LOWSPEED = 0,
то устанавливается высокая скорость (100кГц).

Настройки на плате:
XP163 установлена
XP159 положение OSC
PD13 установлена
PD14 установлена
PF13 установлена
PF14 установлена
PB0 положение OUTA1_1
PB1 положение OUTB1_1
PB2 положение OUTA2_1
PB3 положение OUTB2_1
PB4 положение OUTA1_2
PB5 положение OUTB1_2
PB6 положение OUTA2_2
PB7 положение OUTB2_2
*/
